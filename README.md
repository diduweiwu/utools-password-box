## 用法

- 复制: 双击密码项即可
- 收藏：鼠标右击密码项即可（收藏后会置顶）
- 编辑/删除: 每一个密码项右侧有功能按钮
- 鼠标项按收藏状态置顶，然后按创建时间倒排序
